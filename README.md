# 🐤 Flappy Bird (Python)

A simple Flappy Bird clone built using Python and pygame.

## 🎮 Gameplay

Control the bird and try to pass through as many pipes as possible without crashing.

## 🕹️ Controls

* **SPACE** → Jump

## 🚀 Features

* Score tracking
* High score saving
* Increasing difficulty over time
* Simple and smooth physics

## 🛠️ Built With

* Python 3
* pygame

## ▶️ How to Run

1. Install pygame:

   ```
   pip install pygame
   ```
2. Run the game:

   ```
   python flappy_bird.py
   ```
